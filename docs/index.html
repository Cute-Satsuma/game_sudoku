1. 总则

本应用主要为单机益智游戏。若不使用需要联网的功能（如有），游戏核心玩法可在设备本地完成。

2. 我们收集的信息

当前版本不要求注册账号。我们不主动收集你的姓名、手机号等用于识别你身份的个人信息以运营本游戏。

为提供游戏功能，应用可能在你的设备本地保存与游戏相关的数据（例如闯关进度、设置偏好）。这些数据默认保留在你的设备上，不会由我们主动上传至服务器，除非你使用的版本包含另行说明的联网功能。

若未来版本接入分析、广告、云存档等第三方服务，我们将在本政策中补充其名称、用途及你的选择权，并在应用内或商店详情中提示。

4. 儿童隐私

若你作为家长认为未成年人向我们提交了个人信息，请通过下方联系方式与我们联系，我们将在合理范围内协助处理。

你可随时通过设备的「应用信息」—「清除存储空间/清除数据」删除本应用在本机保存的数据；也可卸载本应用以移除本地数据。
