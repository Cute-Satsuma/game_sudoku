# 数独-可爱桔 - 应用宝市场发布说明书

## 一、账号申请

### 1.1 注册入口

- **腾讯开放平台**：https://open.tencent.com/
- **应用宝开发者平台**：https://app.open.qq.com/

建议使用 **QQ 账号** 登录，点击「应用开放平台」→「管理中心」进入。

> 提示：建议使用公司公共 QQ 号注册，避免员工离职后账号无法管理。

### 1.2 选择开发者类型

注册后需选择 **个人** 或 **企业** 开发者：

| 类型 | 资质要求 |
|------|----------|
| **个人开发者** | 年满 18 周岁且不超过 65 周岁；需人脸校验（通过「腾讯应用宝」公众号扫码）；提供正面手持身份证照片 |
| **企业开发者** | 营业执照（正/副本）彩色扫描件；在营业执照上手写「用于注册腾讯移动开放平台开发者账号+日期」并加盖公章；需显示最新年检情况和统一社会信用代码 |

### 1.3 填写资料

- 公司名称（企业需与营业执照完全一致）
- 统一社会信用代码（企业）
- 联系人姓名、手机号、邮箱
- 详细联系地址
- 银行信息（可选；如需接入支付功能则必填）

### 1.4 实名认证

- **个人**：通过人脸识别完成实名认证
- **企业**：通过银行打款验证完成认证

认证通过后即可创建应用并发布。

## 二、发布流程

### 2.1 创建应用

1. 登录 [应用宝开发者平台](https://app.open.qq.com/)
2. 点击「创建应用」
3. 选择应用类型：**软件**（本应用为数独游戏，选择「软件」或「游戏」下的休闲益智类）
4. 填写应用基本信息：
   - 应用名称、子分类
   - 运营方、研发方
   - 应用简介（60–500 字）

### 2.2 准备并上传资料

| 材料 | 要求 |
|------|------|
| **应用图标** | 1 张，512×512 像素，200KB 以内，PNG 格式，**直角图标**（上传成功后平台将统一规范化处理） |
| **应用截图** | 4–5 张，建议尺寸 1080×1920px，最小不低于 320×480px；JPG/PNG 格式，单张不超过 1M。用于应用宝终端、PC 应用宝、手机管家等应用市场详情页展示 |
| **APK 安装包** | 已用 release 证书签名，无病毒、无恶意代码 |
| **测试账号** | 如应用需要登录，需提供测试账号 |

### 2.3 填写应用详情

- 应用功能特点、更新日志
- 使用权限说明
- 应用分类（游戏 → 休闲益智 / 棋牌游戏）

### 2.4 提交审核

1. 确认所有信息无误后点击「提交审核」
2. 审核周期通常为 **1–7 个工作日**
3. 审核通过后应用自动上架
4. 若被拒，根据反馈修改后重新提交

### 2.5 注意事项

- 建议使用 QQ 浏览器提交，并清除缓存，避免提交失败
- 应用名称、图标、截图需符合平台规范，不含违法违规内容
- 应用包必须有效且通过安全检测

## 四、应用简介

### 一句话描述（15 字以内）
免费无广告，999关数独闯关

### 短描述（80 字以内）
免费无广告！一款简洁好玩的数独益智游戏，支持自由模式与闯关模式。999 关精心设计，三档难度循序渐进，即时反馈助你快速纠错，内置规则与操作指南，支持 18 种语言，随时随地锻炼逻辑思维。

### 详细描述（用于应用详情页）

**免费无广告！经典数独，益智烧脑，随时随地挑战你的逻辑思维！**

数独游戏是一款界面简洁、操作流畅的益智类游戏。无论你是数独新手还是资深玩家，都能在这里找到适合自己的挑战。

**【核心玩法】**
- **自由模式**：自由选择简单、中等、困难三档难度，无限畅玩，想玩就玩
- **闯关模式**：999 关精心设计的数独谜题，从易到难循序渐进，挑战你的极限
- **进度自动保存**：闯关进度自动保存，随时暂停、随时继续，不丢进度

**【贴心功能】**
- **即时反馈**：填入数字后立即显示正确（绿色）或错误（红色），快速纠错，提升效率
- **计时器**：记录每局完成用时，挑战更快的解题速度
- **规则与指南**：内置数独规则说明与操作指南，新手也能快速上手

**【多语言支持】**
支持中文、英语、日语、韩语、法语、德语、西班牙语、俄语等 18 种语言，全球玩家都能畅玩。

**【适合人群】**
- 喜欢逻辑推理与数字谜题的玩家
- 通勤、排队时想打发时间的用户
- 希望锻炼大脑、保持思维活跃的人群

免费下载，无广告干扰，专注纯粹的益智体验！
